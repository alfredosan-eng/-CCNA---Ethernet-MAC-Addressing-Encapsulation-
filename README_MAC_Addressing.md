# 📡 CCNA - Ethernet MAC Addressing & Encapsulation (Chapter 9)

## 📘 Overview
This repository contains study material for Ethernet MAC addressing and encapsulation based on Cisco CCNA.

Ethernet uses MAC addresses to uniquely identify devices in a LAN and relies on frame encapsulation to transmit data between hosts.

---

## 🎯 Topics Covered
- MAC address structure (48-bit)
- OUI (Organizationally Unique Identifier)
- Hexadecimal addressing
- Frame encapsulation
- Layer 2 vs Layer 3 addressing
- Unicast, multicast, and broadcast communication
